# <lo-sample/> 88218

Поняв принципы, по которым составлены таблички чисел, изображённые на рисунках, в первую табличку вставьте недостающее число, а из второй уберите лишнее число.

![]({problemnum}A.png)

## Solution

В первой табличке на каждой строке в первом столбце стоит основание степени, во втором – результат возведения в степень. Таким образом, недостающее число 49.
Вторая табличка построена по-другому: в ней собраны пары равных чисел, но один раз число записано в виде десятичной дроби, другой раз – в виде обыкновенной. Таким образом, здесь лишнее число $\frac{5}{13}$.

## Answer

49; $\frac{5}{13}$.
