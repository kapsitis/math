# <lo-sample/> 65328

Аня ждёт автобус. Какое событие имеет наибольшую вероятность?
$A = \{\text{Аня ждёт автобус не меньше минуты}\}$
$B = \{\text{Аня ждёт автобус не меньше двух минут}\}$
$C = \{\text{Аня ждёт автобус не меньше пяти минут}\}$

## Solution

Событие $A$ включает в себя событие $B$, а событие $B$ включает в себя событие $C$, то есть событие $A$ наиболее обширное: $C \subset B \subset A$. Следовательно, $P(C) \leq P(B) \leq P(A)$.

## Hint 

## Answer 

Событие $A$.
