Remarketing: Mechanisms Combined

A consumer visits your Website \(A\).
      \(\Rightarrow\) It contains a tracking pixel from Website \(B\).
      \(\Rightarrow\) Both Websites \(A\) and \(B\) track the user with the same identifier.
      \(\Rightarrow\) User navigates to Website \(C\) containing Google AdSense.
      \(\Rightarrow\) User is reminded to return to Website \(A\).

Sentiment Analysis Basics

Sentiment Analysis in Latvian

  • For English language there are some Machine Learning models that are already trained to find the sentiments (the model has to be trained by sample data).
  • In Latvian we use a different approach: dictionaries with word stems expressing emotions.
  • Before we analyze anything, all text in user comments is space-normalized (single spaces between any two words, no other punctuation) and converted to the lower case.
